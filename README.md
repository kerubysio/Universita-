# Appunti di Cognitive Computing Systems 🧠

Repository dedicato alla raccolta e all'organizzazione degli appunti per il corso di **Cognitive Computing Systems**. 

## 📝 Metodologia di lavoro
Il materiale contenuto in questa cartella non è una semplice trascrizione delle slide, ma il risultato di un workflow di studio integrato:
* [cite_start]**Analisi delle Slide**: Studio dei diagrammi e dei concetti tecnici ufficiali[cite: 5, 57, 120].
* **Videolezioni**: Approfondimento dei passaggi logici spiegati a voce durante il corso.
* [cite_start]**Supporto AI (Gemini)**: Utilizzo dell'intelligenza artificiale per espandere i punti sintetici delle slide in un discorso fluido e discorsivo, facilitando la comprensione di architetture complesse come **DeepQA** e i microservizi di **Watson**[cite: 57, 107, 161].
* [cite_start]**Obsidian**: Organizzazione ipertestuale delle note per mantenere i collegamenti tra i diversi moduli del sistema (Percezione, Decision Making, Action)[cite: 123, 120, 115].

## 🚀 Come utilizzare gli appunti
Per una visualizzazione ottimale dei grafici, dei collegamenti tra le note e delle immagini delle architetture, si consiglia l'uso di **Obsidian**.

### 1. Clonare il repository
Apri il terminale, recati nella cartella dove desideri salvare il progetto e digita:
```bash
git clone https://github.com/kerubysio/Universita-.git
```

### 2. Configurazione su Obsidian
1. Scarica l'app [Obsidian](https://obsidian.md/).
2. Nella home (in basso a sinistra), clicca sul nome del tuo vault attuale o su **"Manage Vaults"**.
3. Seleziona **"Open folder as vault"**.
4. Scegli la cartella appena clonata dal repository.
